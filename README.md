# SoftGrowTech_expenseTracker_JamesMuriithi
## 🚀 Features
- **Expense Tracking:** Log daily spending with titles, amounts, and dates.
- **Categorization:** Organize expenses (Food, Rent, Transport, etc.) for better insights.
- **Responsive Dashboard:** Built with Tailwind CSS for mobile and desktop access.
- **Persistent Storage:** Uses PostgreSQL for reliable data management.

## 🛠️ Tech Stack
- **Backend:** Python / Django
- **Database:** PostgreSQL
- **Frontend:** HTML5 / Tailwind CSS
